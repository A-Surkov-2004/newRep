Пакеты прикладных программ для статистической обработки и анализа данных

Лекциия 1

Приложения для ведения опорных конспектов. Критерии, которым должны удовлетворять сервисы для создания опорных конспектов:

  • Удобство использования. Сервис должен быть интуитивно понятным и легким в использовании.

  • Организация информации. Сервис должен предоставлять возможности для структурирования информации, создания разделов и подразделов, а также возможность перемещения и переупорядовачивания элементов.

  • Разнообразие элементов. Сервис должен позволять вставлять различные элементы, такие как:

    • текст
    • изображения
    • графики

    • таблицы

      Чтобы опорный конспект был информативнее и нагляднее

  • Возможность совместной работы. При выполнении групповых проектов, или при совместной подготовке, должна быть возможность совместного редактирования, и, отдельно, совместного доступа к документам.

  • Совместимость и экспорт. Возможность импорта и экспорта данных в различных форматах обеспечивает совместимость с другими приложениями и упрощает обмен информацией
  • Автоматическое сохранение. Предотвращает потерю информации в случае сбоя.

  • Бесплатность и доступность. Сервис должен быть либо полностью бесплатным, либо иметь бесплатный уровень доступа

  • Важно чтобы сервис был доступен онлайн без необходимости установки дополнительного ПО.

  • Возможность совместной работы. При выполнении групповых проектов, или при совместной подготовке, должна быть возможность совместного редактирования, и, отдельно, совместного доступа к документам.

  • Инструменты форматирования текстов, заголовков, использование разных шрифтов и стилей оформления. А также форматирование других объектов.

  • Возможность создания диаграмм и ментальных карт. Это полезно для визуализации связи между понятиями

  1. Evernote

    Функционал Это приложение для заметок и организации информации. Предоставляет различные способы создания аннотаций, и добавления комментариев к заметкам Функционал:

    • добавление комментариев к тексту. Можно выделить часть текста в заметке и добавить комментарий или аннотацию к этой выделенной части.

    • Комментарии к изображениям и .pdf (Если прикреплено изображение или .pdf файл к заметке, то Evernote позволяет добавлять аннотации, стрелки, круги и др. графические элементы непосредственно к изображениям и документам. Помогает выделить важную информацию).

    • Аудио и видео аннотации. Можно записывать аудио и видео заметки, а затем добавлять комментарии или аннотации к ним. Это полезно для записи звуковых заметок и комментариев к видео-материалам.

    • Заголовки и форматирование Приложение предоставляет множество инструментов для форматирования текста. Например: заголовки разного уровня. Разное начертание, жирность, курсив, подчеркивание, и др.

    • Теги и метки. Для упорядочивания можно добалять теги и метки.

    • Поиск и фильтрация. Приложение предоставляет мощные инструменты для поиска, включая возможность поиска по тексту аннотаций

  2. Notion

    Функционал

    Многофункциональное приложение для организации информации и управления проектами. имеет широкий спектр возможностей, удобен для создания опорных конспектов и более обширных заметок.

    • В приложении можно создавать блокноты, и внутри них создавать страницы. Т.е создавать разные блокноты для разных тем и проектов.

    • Можно встраивать код, видеоролики, картинки, и тд.

    • Связь и вложенность. Можно создавать связи между страницами. Вложенные страницы помогают структурировать информацию более подробно.

    • Шаблоны. множество готовых шаблонов для различных типов задач включая заметки, планы проектов,

  3. Obsidian

    Функционал

    приложение для создания и организации заметок которое привлекает внимание многих пользователей благодаря своей гибкости и знаниевого управления.

    • Методология заметок. Приложение строит свою методологию на основе заметок, создавая связи между ними, чтобы помочь организовать информацию. Это называется "Связными заметками". Позволяет создавать сеть знаний где каждая заметка может быть связана с другой.
    • Редактор Markdown. Используется для форматирования текста в заметках. Позволяет создавать структурированые и читаемые заметки с использованием простого текстового синтаксиса.
    • Граф знаний. Визуальный граф знаний показывает взаимосвязи между заметками. Это помогает легко находить связанные концепции и идеи, что особенно полезно при исследовании и разработке сложных тем.

  4. Hucidchart

    Функционал Это онлайн сервис для создания диаграмм, схем, орг. карт, и других графических опорных конспектов.

    • Интерфейс для создания диаграмм. Интуитивный и легко понимаемый. Позволяет создавать различные виды диаграмм и схем такие как:

      • Организационные диаграммы
      • Диаграммы потоков работы
      • ER диаграммы
      • Диаграммы Ганта
      • И многие другие...

      Библиотека форм и элементов. Приложение предоставляет богатую библиотеку стандартных форм, символов и элементов, которые можно использовать для создания диаграмм. Также есть возможность загрузки собственных изображений

  5. Simplecture

    Функционал Простое и минималистичное приложение для создания текстовых заметок и организации информации

    • Простата использования. Приложение своей минималистичностью и интуитивной понятностью делает создание и редактирование.

    • Метки. Можно организовать заметки с помощью меток и быстро находить нужную информацию с помощью функции поиска.

    • История версий. Приложение сохраняет историю версий заметок. Что позволяет восстановить предыдущие версии при внесении
